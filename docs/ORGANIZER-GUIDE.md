## מה אתה שולט בו

הכל ידני ובשליטתך המלאה - אין חיבור לשום שירות חיצוני. אתה קובע:
- שם האפליקציה, מי בצוות, מי החתן/ה, התאריכים והיעד.
- כל מילה שמופיעה במסך - בשפה ובהומור של החבורה שלך.
- אילו תמונות עולות (צוות, יעד, ודמויות למשחק).
- מתי המוצר נפתח (מיד / כהפתעה ביום הטיסה).

## הקמה ב-5 דקות

(הסבר טכני מלא ב-[README](../README.md). בקצרה:)

```bash
cp config.sample.php config.php
```

ערוך את `config.php` (פירוט בהמשך), העלה לשרת PHP, ודאג שהתיקיות `data/` ו-`uploads/`
ניתנות לכתיבה. בסיס הנתונים נבנה לבד בכניסה הראשונה.

## config.php - כל ההגדרות

פתח את `config.php`. כל שורה מוסברת בקובץ עצמו. העיקריות:

| הגדרה | מה זה |
|---|---|
| `APP_NAME` | שם האפליקציה. "הפרלמנט"? "החבר'ה"? "הנבחרת"? שלך. |
| `LOGIN_USERNAME` | שם המשתמש המשותף בכניסה (כולם מקלידים אותו + הסיסמה האישית). |
| `HOTEL_URL` | קישור Google Maps ליעד (לכפתור הניווט). |
| `TRIP_START` / `FLIGHT_TS` | תאריך תחילת הטיול ושעת ההמראה. מהם מחושב השעון לאחור והמעבר בין השלבים. |
| `UNLOCK_TS` | רגע פתיחת המוצר המלא (אם בחרת מצב הפתעה). |
| `APP_RELEASED` | `true` = פתוח מיד. `false` = מסך "טיזר" נעול עד `UNLOCK_TS` (להפתעה). |
| `RATING_PARAMS` | פרמטרי הדירוג היומי (מצב רוח, אלכוהול...). שנה לטעמך. |
| `PACKING_ITEMS` | רשימת "לא לשכוח" ההתחלתית. |
| `USERS` | הצוות - ראה למטה. |

**שינוי תאריכים:** השורות נראות כך - שנה את המספרים (חודש, יום, שנה):
```php
define('TRIP_START', mktime(0, 0, 0, 7, 1, 2026));  //  7=חודש, 1=יום, 2026=שנה
```

## הצוות והתמונות

ב-`config.php`, בתוך `USERS`, כל חבר הוא שורה כזו:

```php
'alon' => ['name' => 'אלון', 'is_groom' => false,
    'password' => '...hash...', 'phrase' => 'מי שארגן את הבלאגן.',
    'tel' => '0500000002', 'since' => 2008, 'tags' => ['אנרגיות'], 'bio' => '...'],
```

- **המפתח** (`'alon'`) = מזהה באנגלית, ללא רווחים. גם שם קובץ התמונה שלו.
- `is_groom => true` רק לאחד - הוא מקבל כתר.
- **סיסמה:** צור עם הפקודה (החלף PASS בסיסמה):
  ```bash
  php -r 'echo password_hash("PASS", PASSWORD_BCRYPT, ["cost"=>12]);'
  ```
- `phrase` = משפט קצר שמופיע מתחת לשם. `bio` = תיאור ארוך בכרטיס. `tags` = תגיות
  (כל חבר יכול לערוך את שלו מתוך האפליקציה).

**תמונת פרופיל:** שים קובץ בשם `assets/members/<מפתח>.jpg`. לדוגמה: `assets/members/alon.jpg`.

## תמונות המשחק והיעד

- **דמויות למשחק "הגנת החתן"** - התמונות ש"נופלות" במשחק. שים אותן בתיקייה
  `assets/women/` (כל קובץ `.jpg`). **לבחירתך** - בנות, בנים, פרצופים מצחיקים, מה שמתאים
  לחבורה ולחתן/ה. המשחק לוקח אוטומטית את כל מה שיש בתיקייה.
- **תמונות היעד/מלון** - שים ב-`assets/hotel/`. מופיעות באזור הניווט.

## המקומות לפי ימים

בדיוק כמו שהחברים רואים: לכל יום יש 2 משבצות לצהריים ו-2 לערב. משבצת ריקה מציגה
קו מקווקו עם "+ הוספת מקום".

- **ברירת מחדל:** המשבצות ריקות, והחברים ממלאים בעצמם לאורך הימים שלפני הטיסה
  (כל אחד מדביק קישור Google Maps למקום שבא לו).
- **רוצה להזין מראש?** אתה (או כל חבר) פשוט מוסיף מקומות מהאפליקציה - הם נשמרים
  לפי יום וארוחה. אין צורך לגעת בקוד.

## כל הטקסטים המותאמים

כאן הקסם. כל משפט באפליקציה אפשר לשנות לשפה של החבורה. פתח את הקובץ, מצא את הטקסט,
החלף את מה שבין הגרשיים. (מומלץ לייצר את כולם בבת אחת עם [המחולל](#-מחולל-הקופי-עם-ai-הכי-כיף).)

| מה | קובץ | סימן לחיפוש |
|---|---|---|
| משפט במסך הכניסה | `index.php` | `הטיול מתחיל כאן` |
| כיתובי הימים (5 ימים, צהריים+ערב) | `app.php` | `DAY_CAPTIONS` |
| משפטי הגלגל המצחיקים | `games.php` | `WHEEL_LINES` |
| הסבר משחק הגלגל | `games.php` | `מסובבים, יוצא מישהו` |
| הודעת "כבר יש את זה" ברשימה | `api/packing.php` | `עוד לא שתינו` |
| הודעת נעילת ניווט | `navigate.php` | `איפה תלך דוד?` |
| הודעת נעילת ברכות | `blessings.php` | `מומלץ להירגע` |

**דוגמה - כיתובי הימים** (`app.php`, חפש `DAY_CAPTIONS`):
```js
const DAY_CAPTIONS = {
    1: { lunch: 'רק נחתתם. לאט לאט.', dinner: 'לילה ראשון. מי שנרדם ראשון - מצויר.' },
    2: { lunch: 'בוקר אחרי. מישהו זוכר מה היה אתמול?', dinner: 'הערב מתחמם.' },
    ...
};
```
פשוט החלף את הטקסטים בגרסה של החבורה שלך.

## מחליף הצבעים

האפליקציה מגיעה עם 4 פלטות (זהב / ורוד / סגול / תכלת), כולן בדארק מוד. כל אחד בוחר
לעצמו מתחתית הבית. ורוד מצוין למסיבת רווקות, למשל. אין מה להגדיר - זה פשוט עובד.

## 🤖 מחולל הקופי עם AI (הכי כיף)

זה החלק שהופך את האפליקציה ל*שלכם*. במקום לחשוב על עשרות משפטים - תן ל-AI (ChatGPT,
Claude, וכו') להכיר את החבורה ולכתוב הכל בסגנון שלהם. **העתק את הפרומפט, מלא את הסוגריים,
הדבק ב-AI, וקבל בחזרה את כל הטקסטים מוכנים להדבקה.**

```
אני מקים אפליקציית טיול לחבורת חברים ואני צריך ממך לכתוב את כל הטקסטים בשפה
ובהומור המדויקים של החבורה. הנה ההקשר:

- שם החבורה: [למשל "הפרלמנט"]
- לאן טסים: [למשל "אתונה, 5 ימים"]
- האירוע: [למשל "מסיבת רווקים", "רווקות", "טיול שנתי"]
- מי החתן/ה: [שם + משהו עליו]
- חברי החבורה ותכונה לכל אחד: [שם - תכונה / תפקיד בחבורה. שורה לכל אחד]
- בדיחות פנימיות / ביטויים שחוזרים אצלנו: [זרוק כמה שיותר - משפטים, כינויים, מקרים מיתולוגיים]
- הטון: [למשל "גברי, שובב, עוקצני, עברית מדוברת. בלי אמוג'ים"]

עכשיו כתוב לי, באותו טון בדיוק:

1. כיתובי ימים - לכל אחד מ-5 הימים, משפט לצהריים ומשפט לערב (10 בסך הכל).
   קצרים, מצחיקים, מתייחסים להתקדמות הטיול והעייפות שנצברת.
2. 14 משפטים אקראיים ל"גלגל" - הוראות צ'יזבט קצרות שמישהו מקבל כשהגלגל נופל עליו
   (למשל "שתה מים. עכשיו.", "מי שמדבר על עבודה - שותה."). אחד מהם יכול להזכיר את החתן בשמו.
3. משפט קצר למסך הכניסה.
4. הודעת "כבר יש את זה" שמופיעה כשמישהו מנסה להוסיף פריט שכבר ברשימת האריזה
   (בסגנון עוקצני).
5. שתי הודעות נעילה קצרות ומצחיקות (אחת ל"ניווט", אחת ל"ברכות") שמופיעות לפני שהטיול התחיל.

תן הכל מוכן להעתקה, מסומן לפי הסעיפים למעלה.
```

קח את התשובה, ולפי [טבלת הטקסטים](#כל-הטקסטים-המותאמים) הדבק כל חלק במקום שלו. זהו -
האפליקציה מדברת עכשיו בשפה של החבורה.
